# Pages

> Localizando a pasta pages no projeto ├─ src/pages

Para criar uma nova página, no projeto, execute o seguinte comando:


```bash

npm run gen NomeDoPage

### Logo após digite P:
? component digite: (C || page digite: P) P

```
